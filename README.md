CRASHBURN
=========
This is the crashburn project, a simple 3D multiplayer video game, 
based on Crash Bash.

BUILDING
========

Dependencies
------------
- libglfw 2.7.7 or greater
- ...

Instructions
------------
Just as any CMake project:

    git clone git://github.com/killstealers/crashburn.git
    mkdir crashburn.build
    cd crashburn.build
    cmake ../crashburn
    make -j
