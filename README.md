CRASHBURN
=========
This is the crashburn projet.

BUILDING
========

Dependencies
------------
- libglfw
- ...

Instructions
------------
Just as any CMake project:

    git clone git://github.com/killstealers/crashburn.git
    mkdir crashburn.build
    cd crashburn.build
    cmake ../crashburn
    make -j
